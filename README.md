# UDTech Dev Platform

UDTech Dev Platform is a collaborative development project designed for interns to gain real-world experience in building full-stack web applications.

This repository serves as the main workspace where developers contribute to building scalable platforms such as event management systems, hackathon portals, dashboards, and developer tools.

## Objectives

- Provide hands-on experience with real-world development workflows
- Teach collaborative development using GitHub
- Build production-ready features under project lead supervision
- Encourage best practices in coding, documentation, and version control

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript
- React.js

Backend
- Node.js
- Express.js

Database
- MongoDB / PostgreSQL

Tools
- Git & GitHub
- REST APIs
- Agile development workflow

## Development Workflow

1. Interns work on assigned issues
2. Create a feature branch
3. Develop the feature
4. Submit a Pull Request
5. Project Lead reviews the code
6. Approved code is merged into the main project

## Contribution Guidelines

- Do not push directly to the main branch
- Always create a feature branch
- Submit Pull Requests for review
- Follow clean code and documentation practices
